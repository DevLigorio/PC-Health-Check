# PC Health Check & Optimization Tool

Advanced Windows maintenance script for diagnostics, cleanup, and optimization.

## Features

- 🔍 System diagnostics
- 🧹 Remove bloatware
- ⚡ Optimize performance
- 🗑️ Clean temp files
- 🔧 Repair system files
- 📦 Update software

## Requirements

- Windows 10 or 11
- Administrator rights

## Installation

1. Download `PC_Health_Check.bat`
2. Right-click → **Run as administrator**
3. Select operations from menu
4. Press **S** to start

## Menu Options

| Key | Operation |
|-----|-----------|
| 1 | System Information |
| 2 | Remove Bloatware & Telemetry |
| 3 | Optimize Services |
| 4 | Registry Tweaks |
| 5 | Clean Temp Files |
| 6 | Reset Network |
| 7 | Repair System Files (SFC/DISM) |
| 8 | Clean Windows Update Cache |
| 9 | Update Software (Winget) |
| 0 | Schedule Disk Check |
| A | Select All |
| S | Start |

## Safety

✅ Creates restore point before changes  
✅ Detailed logging to Desktop  
✅ All operations are reversible  

## What Gets Removed (Option 2)

Xbox apps, Bing apps, 3D Builder, Skype, Candy Crush, and 20+ other bloatware apps.

## Output

Logs saved to Desktop:
- `PC_Health_Report_[timestamp].txt` - Full log
- `PC_Health_Errors_[timestamp].txt` - Errors only
